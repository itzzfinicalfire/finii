# Infinitely drawing icons

A Pen created on CodePen.io. Original URL: [https://codepen.io/ainalem/pen/dKjgBx](https://codepen.io/ainalem/pen/dKjgBx).

Using SVG line animation to infinitely draw the outline of connected icons. This outline together with a couple of additional lines create a loader which cycles icons.

Vector icons from freepik.com
